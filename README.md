# ToDoList
Данная программа реализует:
1. Создание базы данных
2. Сохранение, редактирование, удаление, данных из базы данных
3. Так же реализация небольшой верстки страницы

Процедура запуска:
1. Установить MYSQL
2. Добавить название новую схему в MySQL, название взят из application.yml
3. Ввести пользователя и пароль, для соединения с БД.
4. запустить программу
5. В адрессной строке браузера ввести http://localhost:8080/
